# Scavenger Hunt App

## Overview

Scavenger Hunt App is an iOS application built using SwiftUI as part of the MWD3A (iOS Development) Assignment 3. The app encourages users to explore local businesses in the Greater Toronto Area by participating in a digital scavenger hunt.

Users receive clues for hidden items located at participating businesses. Once an item is found, the user can select a proof photo using the PhotosPicker feature. Progress is tracked throughout the hunt, and rewards are granted based on the number of items discovered.

---

## Features

- Home screen with scavenger hunt information
- List of 10 participating local businesses
- Individual hunt detail screens with clues
- Image assets for each business
- Photo selection using PhotosPicker
- Progress tracking system
- Reward system based on hunt completion
- Navigation using NavigationStack and NavigationLink
- Shared app data using ObservableObject and EnvironmentObject
- Custom SwiftUI views and layouts

---

## Reward System

- Find 5 items → 10% Discount Code
- Find 7 items → 20% Discount Code
- Find all 10 items → 20% Discount Code + Grand Prize Entry
